Chief of Staff System

OpenAI · Zapier · Google Forms · Google Sheets · Notion · Workflow Automation

Project Overview

Built an AI-powered execution system using OpenAI that transforms messy inputs into structured plans, tasks, and priorities in Notion. The workflow captures ambiguous project context, turns it into a clean execution plan, and stores the output in a centralized workspace for visibility and follow-through.

Workflow

Google Form → Google Sheets → Zapier → OpenAI → Code by Zapier → Notion

What the System Does

  • Captures unstructured project or operations inputs through a simple intake form.
  • Uses OpenAI to generate a project title, goals, tasks, risks, priorities, and next steps.
  • Parses structured output and maps it into clean Notion database fields.
  • Creates a reusable execution layer that turns ambiguity into organized action.

Why I Built It

Teams often lose momentum because important context lives in scattered notes, chat messages, or founder brain dumps. I built this system to show how AI can act like an execution partner, converting messy inputs into a consistent operating structure that supports alignment and faster follow-through.

Key Outcome

This project demonstrates AI-assisted execution design: taking unstructured inputs, enforcing structure, and turning them into a workflow that is easier to track, prioritize, and act on.
