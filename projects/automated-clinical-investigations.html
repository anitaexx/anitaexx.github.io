Power Automate – Workflow Automation

Power Automate · Excel · SharePoint · Microsoft Forms

Overview

The clincal team are constantly going back and forth with clients for information on clinical investigations. This solution offers a Microsoft Form for clients to fill, with fields they must complete, allowing uploads. With Power Automate the form's responses are extrated to an Excel file and uploads saved in a SharePoint folder. I was able to automate creating and populating excel files, downloading files and creating folders in SharePoint. Each submission has it's own folder.

Key Outcomes

  • Reduced manual processing by ~90%
  • Improved turnaround time by ~50%
  • Standardized data capture and validation
