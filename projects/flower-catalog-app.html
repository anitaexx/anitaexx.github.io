Project Overview

This project is a Microsoft PowerApps Canvas App designed for a florist or botanist to manage flower inventory through a simple, user-friendly interface.

The application allows users to add new flowers to a centralized SharePoint-backed database, edit existing records, and maintain accurate, up-to-date inventory information without technical expertise.

SharePoint serves as the data layer, while PowerApps handles the front-end logic, validation, and user interaction. Please refer to the video above for a full walkthrough of the app’s functionality and design.
